UX|UI Design

Tanto em User Experience (UX) quanto User Interface (UI), a excelente experiência do usuário/cliente é a chave-mestra. A área de UX trata de como uma pessoa se sente enquanto usa qualquer produto ou serviço, digital ou não, oferecido pela sua empresa e tem a ver com a sensação dessa pessoa.

HTML

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS|BOOTSTRAP

CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web..

JAVASCRIPT

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.
